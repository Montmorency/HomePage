Text Editors

For editing text files, and doing developement I like to use VIM. I typically use the plainest vanilla configuration of vi available typically only reconfiguring the default tab style to two white spaces. Here is a useful article for getting your environment set up with more bells and whistles: VIM Environment Configuration. Here are the core lines I like to add to ~/.vimrc whenever I'm on a new computer. syntax on au BufNewFile, BufRead *.py set tabstop=2 set softtabstop=2 set shiftwidth=2 set textwidth=140 set expandtab set autoindent set fileformat=unix set ruler

SSH

In your ~/.ssh/config file it is a good idea to set up Host aliases. This makes using rsync and scp between machines a lot simpler. The typical pattern is below.

Host * ServerAliveInterval 240 Host nameofcomputer Hostname comp.at.another.location.uk User myusername Port 22

General Configurations can use pattern matching i.e. Host * to set configurations across every server we plan on connecting to. Here we are specifying how often to ping the remote server ServerAliveInterval 240 . so that the connection stays alive (every 4 minutes in this case). Subsequent specific aliases detail. By default ssh processes listen on port 22.
